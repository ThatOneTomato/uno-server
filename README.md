# Messenger Server

A **simple messenger server** built with JavaScript for websites, with extensive upcoming features!

## Features
- A robust backend written 100% in JavaScript.
- Supports messaging functionality with plans for future expansion.
- Easy integration with web-based frontends.

Upcoming features:
- Expanded user authentication and security measures.
- Real-time messaging and notifications.
- Group chat functionality.
- Media file sharing capabilities.

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ThatOneTomato/uno-server.git
   cd uno-server
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the server:
   ```bash
   node server.js
   ```

   By default, the server will run on `http://localhost:3000`.

## Usage
- Integrate your frontend application with this server to enable messaging functionality for your users.
- Configure the server to add desired features according to your project's requirements.
